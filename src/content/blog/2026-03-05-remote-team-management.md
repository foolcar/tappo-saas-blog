## 背景：為什麼需要遠程團隊？

當Tappo決定出海時，我們面臨的第一個組織挑戰是：**我們不可能在每個目標市場都設立全職辦公室。**

香港的辦公成本已經很高（平均每呎HK$50+），如果在曼谷、雅加達、新加坡都租辦公室，運營成本會吃掉大部分毛利。

解決方案是組建一支跨國遠程團隊——核心團隊在香港，本地團隊在目標市場遠程工作。

## 我們的團隊結構

```
                香港總部 (10人)
              產品 │ 技術 │ 運營
                 │
    ┌────────────┼────────────┐
    │            │            │
新加坡(2人)   泰國(3人)    印尼(2人)
銷售+客戶成功  銷售+支援   銷售+BD
```

**核心原則：** 產品和技術留在香港，銷售和客戶成功在本地。

原因很簡單：產品開發需要密集協作，時區越近越好；銷售必須懂當地語言和文化，必須在當地。

## 招聘：找到對的人比什麼都重要

### 在哪裡找人？

| 渠道 | 適合職位 | 效果 |
|------|----------|------|
| LinkedIn | 銷售經理、BD | ★★★★ |
| 當地招聘網站（如JobsDB、JobStreet） | 客戶成功、技術支援 | ★★★ |
| Upwork / Toptal | 短期項目、兼職開發 | ★★★ |
| 行業展會/活動 | 高級銷售、合作夥伴 | ★★★★ |
| 個人網絡推薦 | 所有職位 | ★★★★★ |

**最重要的招聘經驗：第一個本地員工必須找對。**

我們在泰國的第一個銷售經理是通過一個香港朋友介紹的——他之前在GrabFood做BD，對餐飲行業非常熟悉。這個人後來成為了我們整個泰國業務的基石。

### 遠程面試的特殊考量

- **書面溝通測試**：讓候選人寫一份英文郵件或Slack消息。遠程團隊70%的溝通是文字，這比口語能力更重要
- **自主性測試**：「描述一個你完全獨立完成、沒有上級指導的項目」
- **時區紀律測試**：「你之前的遠程工作經驗中，如何管理不同時區的會議？」

## 跨時區協作：工具和方法

### 我們的工具棧

| 工具 | 用途 | 費用/月 |
|------|------|---------|
| Slack | 日常溝通 | HK$70/人 |
| Notion | 文檔、知識庫 | HK$80/人 |
| Linear | 任務管理 | HK$65/人 |
| Loom | 異步視頻更新 | HK$100/人 |
| Zoom | 視像會議 | HK$120/人 |
| Deel | 跨國薪資合規 | HK$500/人 |

**關鍵洞察：** 異步溝通 > 實時會議。不要試圖讓所有人在同一時間在線——這對跨時區團隊來說是不可能的。

### 會議紀律

我們制定了嚴格的會議規則：

1. **周一/五全員站會**（30分鐘）：香港時間10:00 AM（新加坡同步，泰國9:00 AM）
2. **周三不開會**：深度工作時間
3. **所有會議必須有Agenda**：提前24小時發在Slack
4. **會議後必須有書面總結**：發在對應的Slack頻道
5. **一對一每兩週一次**：30分鐘，由員工主導

### 異步日報

我們不用傳統的日報（太形式化），而是用一個簡單的Slack格式：

```
📋 今日完成
• 簽約了2家曼谷餐廳（Sukhumvit區）
• 解決了客戶A的打印機問題

🔜 明日計劃
• 跟進3個潛在客戶Demo
• 和技術團隊討論泰語翻譯質量

🚧 需要幫助
• 泰語UI翻譯質量需要review（對應Linear ticket #234）
```

## 文化差異：不是翻譯問題

管理跨國團隊最大的挑戰不是語言，而是工作文化差異：

| 文化維度 | 香港/中國 | 泰國 | 印尼 | 建議 |
|----------|----------|------|------|------|
| 直接性 | 直接 | 間接 | 非常間接 | 注意語氣 |
| 等級觀念 | 中 | 高 | 高 | 尊重頭銜 |
| 反饋方式 | 公開批評可接受 | 需要私下說 | 必須非常婉轉 | 一對一給反饋 |
| 時間觀念 | 較準時 | 較彈性 | 彈性 | 設定明確期望 |
| 節假日 | 中西混合 | 佛教節日 | 伊斯蘭節日 | 共享假期日曆 |

**一個血淚教訓：** 我們早期在泰國團隊推行「公開code review，直接指出問題」的文化，結果泰國同事覺得非常不受尊重，離職率高達50%。後來改成「私下code review + 團隊分享會」，離職率降到零。

## 績效考核：如何評估遠程團隊

我們放棄了傳統的「工時考核」，轉向**結果導向的OKR系統**：

```yaml
銷售經理 - 泰國:
  Objective: 提升泰國市場客戶數
  Key Results:
    - KR1: 簽約20家新餐廳
    - KR2: 客戶留存率 > 85%
    - KR3: 平均客戶滿意度 > 4.0/5.0

客戶成功 - 新加坡:
  Objective: 提升客戶健康度
  Key Results:
    - KR1: 每月活躍使用率 > 80%
    - KR2: 平均響應時間 < 2小時
    - KR3: NPS分數 > 50
```

**核心原則：OKR必須是結果，不是活動。**「打了100通電話」是活動，「簽約5家餐廳」才是結果。

## 團隊建設：遠程不遠心

遠程團隊的最大風險是疏離感。我們的策略：

1. **季度線下聚會**：每個季度選一個市場（曼谷/新加坡/香港輪換），全員聚會3天
2. **虛擬茶水間**：Slack開一個 #random 頻道，只聊非工作話題
3. **生日/節日驚喜**：通過當地外賣平台給遠程同事送蛋糕/禮物
4. **在線遊戲之夜**：每月一次線上桌遊/遊戲

## 結論

遠程團隊不是省錢的手段，而是獲取全球人才的策略。管理得好的遠程團隊，生產力可以超過同地辦公室團隊——因為每個人都在最舒適的環境中工作。

> 阿公寄語：管理遠程團隊最緊要係信任。你請人嗰陣要信佢搞得掂，畀佢自由發揮。如果唔信佢，不如一開始就唔好請。
